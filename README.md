# eBid - Collectibles Auction Website

![Screenshots](./public/screenshot.png)

Welcome to the eBid repository, an online collectibles auction website designed and developed as part of a project for the "Desenvolvimento de Software para Web 2" (Web Software Development 2) course at UFSCar (Federal University of SÃ£o Carlos). This project aims to create an engaging and interactive platform for collectibles enthusiasts, facilitating the buying and selling of exclusive and rare items through real-time auctions.

## Technologies Used

- **HTML/CSS:** Responsible for the structure and styling of the website, providing an aesthetically pleasing and user-friendly interface.

- **JavaScript:** Utilized for implementing interactive and dynamic features, enhancing the overall user experience by allowing real-time bidding and updates.
